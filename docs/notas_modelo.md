# Notas del modelo inicial

## Objetivo

Construir una primera versión manual del modelo de propagación de fuego.

## Variables de entrada

- Velocidad del viento
- Dirección del viento
- Pendiente del terreno
- Tipo de combustible
- Tiempo de propagación

## Salida esperada

- Velocidad estimada de avance
- Distancia estimada recorrida por el frente del fuego
- Dirección principal de propagación

## Criterio de esta primera fase

No usamos todavía mapas, APIs ni datos externos.

Primero queremos que el cálculo funcione de forma sencilla y comprobable.
